# String常用API

## #String字符串分割数组
`str.split(",")`
## #判断string是否以 "xxx" 开头
`str.startsWith("xxx")`
## #判断string是否含有"xxxx"字符串
`str.contains("xxx")`
## #将"aaa"代替为"xxx"
`str.replace("aaa","xxx")`
## #从当前位置截到末尾
`str.substring(0)`
## #截取内容(包前不包后a=<x<b)
`str.substring(0,3)`
## #字符串转换为字符数组
`str.toCharArray()`
