-c # 统计字节数，或--bytes或——chars：只显示Bytes数；。
-l # 统计行数，或——lines：只显示列数；。
-m # 统计字符数（空格也算一个字符）。这个标志不能与 -c 标志一起使用。
-w # 统计字数，或——words：只显示字数。一个字被定义为由空白、跳格或换行字符分隔的字符     串。 
-L # 打印最长行的长度。

通常结合ls使用比如 ls -1分行输出，
统计文件个数通过 wc - l
